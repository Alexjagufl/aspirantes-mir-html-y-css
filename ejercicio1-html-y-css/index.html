Receta: Ensalada

Ingredientes

Los ingredientes necesarios para preparar la ensalada son:

  • Pechuga
  • Lechuga
  • Tomate
  • Cebolla
  • Maíz

Preparación
  1. Se levan y desinfectan los ingredientes
  2. Se pican todos los ingredientes
  3. Se mezclan los ingredientes
  4. Listo puedes degustar
